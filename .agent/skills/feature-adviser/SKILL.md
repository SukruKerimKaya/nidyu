# Feature Adviser Yönergeleri

1. Kullanıcının özellik isteğini anla.
2. Mevcut sistem mimarisine (Design System, Atomic Design) uygunluğunu kontrol et.
3. Çözüm önerisi sun ve koda dökmeden önce onaya sun.
